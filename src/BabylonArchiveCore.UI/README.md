# BabylonArchiveCore.UI

Слой представления — всё, что видит и с чем взаимодействует игрок.

## Ответственность
- HUD (здоровье, ресурсы, мини-карта)
- Экраны (меню, инвентарь, диалоги, карта Архива)
- Обработка ввода (клавиатура, мышь, геймпад)
- Доступность (accessibility)
- Локализация
- Debug-оверлеи

## Зависимости
- **Зависит от**: `BabylonArchiveCore.Core`, `BabylonArchiveCore.Runtime`
- **Запрещено**: прямая мутация WorldState — только через Runtime-команды

## Поддиректории (будут созданы в последующих сессиях)
```
HUD/             — игровой интерфейс
Screens/         — экраны и меню
Input/           — обработка ввода
Accessibility/   — доступность
Localization/    — локализация
Debug/           — debug-оверлеи
```
