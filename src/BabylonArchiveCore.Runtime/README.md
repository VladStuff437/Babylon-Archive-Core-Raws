# BabylonArchiveCore.Runtime

Игровой цикл и механики — связывает доменные модели с игровым процессом.

## Ответственность
- Игровой цикл (GameLoop)
- Боевая система (Combat)
- ИИ (AI)
- Процедурная генерация (Generation)
- Система миссий (MissionRunner)
- Система событий (EventBus)
- Аудио-менеджер
- Управление сценами

## Зависимости
- **Зависит от**: `BabylonArchiveCore.Core`
- **Запрещено**: ссылки на UI

## Поддиректории (будут созданы в последующих сессиях)
```
GameLoop/        — основной цикл
Combat/          — боевая система
AI/              — искусственный интеллект
Generation/      — процедурная генерация, seed
Missions/        — MissionRunner, валидация
Events/          — EventBus
Audio/           — аудио-менеджер
Scenes/          — управление сценами
```
