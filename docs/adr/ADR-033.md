# ADR-033: Loot System Consolidation

## Status
Accepted

## Context
Session 033 formalizes loot rarity tables and runtime drop resolution while keeping deterministic behavior with weighted outcomes and optional luck bias.

## Decision
- Introduce Session033 loot contract/schema/migration/serializer.
- Extend DropResolver with tier resolution (`ResolveDropTier`) and deterministic drop selection (`ResolveDrop`).
- Add session-033 balance profile with rarity weights and luck bias.
- Extend mission/domain validation utilities to support stronger integration tests.

## Consequences
- Loot generation is data-driven and easier to balance per-session.
- Runtime keeps deterministic roll semantics while allowing tuning by profile.
