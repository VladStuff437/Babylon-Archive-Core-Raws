# ADR-035: Faction Reputation and World Axis Integration

## Status
Accepted

## Context
Session 035 finalizes faction reputation and world-axis effects as first-class runtime factors influencing mission outcomes and combat context scaling.

## Decision
- Introduce Session035 faction reputation contract/schema/migration/serializer.
- Extend WorldState and WorldStateService with axis setting and reputation delta APIs.
- Extend DamageCalculator with context-aware scaling (reputation + world axes).
- Add session-035 balance profile and control profile.

## Consequences
- Faction/world alignment can influence runtime systems without ad-hoc hooks.
- Mission/economy/combat layers share a consistent state interpretation.
