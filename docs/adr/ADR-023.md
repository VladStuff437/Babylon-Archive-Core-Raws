# ADR-023: Inventory V1 Consolidation (S023)

## Status
Accepted

## Context
S023 consolidates inventory v1 functionality: formal contract, schema, serialization round-trip, migration, and integration with the gameplay loop established in S022.

## Decision
- Add `Session023InventoryContract` + `InventorySlot` in Core.
- Add `session-023.schema.json` for state validation.
- Add `Session023Serializer` and `Migration_023`.
- Add `session-023.profile.json` for inventory controls.
- Extend `SessionEvents` with `Session023InventoryV1Ready`.
- Full runtime + smoke tests covering inventory CRUD, combat, AI, perception in one gameplay loop.

## Consequences
- Inventory system is testable and serializable end-to-end.
- Enables character attributes (S024) to consume items for stat effects.
