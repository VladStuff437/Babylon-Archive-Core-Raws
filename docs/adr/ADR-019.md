# ADR-019: Input Action Map with Fallback

**Статус**: Accepted
**Дата**: Сессия 019
**Контекст**: ADR-018

## Решение
Ввести контракт Session019ActionMapContract и каталог ActionMapCatalog с fallback-резолвом действий ввода.

## Альтернативы
1. Жёстко кодировать бинды в CommandPipeline: отклонено.
2. Использовать только один control profile без fallback: отклонено.

## Риски
- Конфликты биндов между профилями и fallback-цепочкой.
- Потребуется приоритизация device-specific слоёв.

## Связи
- Реализует: INV-024
- Зависит от: ADR-018
- Влияет на: S020+
