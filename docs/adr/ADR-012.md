# ADR-012: Схема сейвов v1

**Статус**: Accepted
**Дата**: Сессия 012
**Контекст**: ADR-011

## Решение
Зафиксировать `Session012SaveContract` и соответствующую JSON-схему как минимальный стабильный save пакет с поддержкой миграций.

## Альтернативы
1. Хранить бинарный формат без JSON-схемы: отклонено, сложнее валидация и диагностика.
2. Жестко связывать save со сборкой: отклонено, нарушает обратную совместимость.

## Риски
- Рост стоимости миграций при быстром расширении полей.
- Возможные несовпадения временных меток между платформами.
