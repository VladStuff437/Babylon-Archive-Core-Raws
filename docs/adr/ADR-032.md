# ADR-032: Baseline Balance v1

## Status
Accepted

## Context
Сессия 032 фиксирует baseline balance profile и синхронизирует его с контуром урона, экономики и миссионных эффектов в WorldState.

## Decision
- Ввести `Session032BalanceContract` + schema/migration/serializer/profile.
- Зафиксировать `session-032.json` как baseline профиль множителей.
- Расширить покрытие runtime/integration тестами S032.

## Consequences
- Баланс v1 формализован как данные и готов к дальнейшей настройке.
- Экономика и боевой урон синхронизированы с единым профилем баланса.
