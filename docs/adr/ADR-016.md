# ADR-016: Minimal Game Loop Scaffold

**Статус**: Accepted
**Дата**: Сессия 016
**Контекст**: ADR-015

## Решение
Ввести минимальный game loop scaffold (tick/pause/resume/update) и связать его с контуром команд и control profiles.

## Альтернативы
1. Реализовать полный game loop сразу: отклонено.
2. Оставить loop только в UI: отклонено, нарушает слой Runtime.

## Риски
- Ограниченный каркас может потребовать рефакторинга на блоке 03.
- При отсутствии метрик loop трудно оптимизировать рантайм.
