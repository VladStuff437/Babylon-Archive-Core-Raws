# ADR-039: Node Transition Evaluation Stabilization

## Status
Accepted

## Context
Session 039 introduces deterministic scoring rules for mission node transitions and balance-driven evaluation parameters.

## Decision
- Introduce Session039 transition-evaluation contract/schema/migration/serializer/profile.
- Extend TransitionEvaluator with explicit scoring parameters and balance-driven selection.
- Add runtime/mission/smoke tests for deterministic scoring and fallback behavior.

## Consequences
- Transition selection remains deterministic under equal-priority and mixed fallback scenarios.
- Balance tables can tune mission transition behavior without runtime code changes.
