# ADR-018: Command Contour Validation

**Статус**: Accepted
**Дата**: Сессия 018
**Контекст**: ADR-017

## Решение
Ввести контракт Session018CommandContourContract и реестр CommandContourRegistry, выполняющий mode-aware валидацию команд до мутации WorldState.

## Альтернативы
1. Валидировать команды только в mutator: отклонено.
2. Валидировать команды через UI фильтры: отклонено.

## Риски
- Переусложнение контура при большом количестве типов команд.
- Требуется договориться о политике для незарегистрированных команд.
