# ADR-045: Mission Runtime Save/Load Integrity

## Status
Accepted

## Context
Session 045 formalizes save/load integrity for mission runtime snapshots with deterministic checksum validation.

## Decision
- Introduce Session045 persistence contract/schema/migration/serializer/profile.
- Extend MissionRuntimeSnapshot with snapshot version and state checksum metadata.
- Validate snapshot checksum during deserialize before state restoration.

## Consequences
- Corrupted or tampered mission snapshots are rejected before restore.
- Mission runtime persistence gains explicit compatibility and integrity guarantees.
