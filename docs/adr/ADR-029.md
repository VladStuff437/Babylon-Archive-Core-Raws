# ADR-029: Enemy AI Perception Baseline

## Status
Accepted

## Context
Сессия 029 формализует перцепцию врагов: детекция по радиусу + awareness score и выбор primary target, а также интеграцию в state machine.

## Decision
- Расширить `PerceptionSystem` методами `DetectTargetsWithAwareness` и `SelectPrimaryTarget`.
- Добавить в `AIStateMachine` метод `UpdateFromPerception`.
- Добавить session-артефакты: contract/schema/migration/serializer/control profile/tests.

## Consequences
- Контур перцепции и реакций ИИ становится явным и проверяемым.
- Подготовлен фундамент для S030 (углубление state machine врагов).
