# ADR-036: World Axis Consolidation

## Status
Accepted

## Context
Session 036 finalizes world-axis behavior as a first-class runtime contract tied to balance profile loading and mission effects.

## Decision
- Introduce Session036 world-axis contract/schema/migration/serializer/profile.
- Extend WorldState and WorldStateService with axis delta + snapshot APIs.
- Integrate balance profile usage into combat/loot/economy runtime paths.

## Consequences
- World alignment now has deterministic APIs and profile-driven tuning.
- Mission and combat systems can consume consistent axis state.
