# ADR-027: Combat Input Safety Lock (Tab/Esc)

## Status
Accepted

## Context
Сессия 027 фиксирует поведение боевого ввода Tab/Esc в реальном времени и исключает побочные эффекты во время миссионных переходов.

## Decision
- Расширить `CombatInputHandler` миссионной блокировкой (`BeginMissionTransitionLock` / `EndMissionTransitionLock`).
- Во время блокировки запретить изменение цели через `TabTarget` и сброс через `EscCancel`.
- Добавить session-артефакты: contract/schema/migration/serializer/control profile/tests.

## Consequences
- Переходы миссий не ломаются из-за конкурентных команд ввода.
- Поведение таргетинга стабильно и предсказуемо в runtime.
