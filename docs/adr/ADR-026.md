# ADR-026: Combat Core Expansion

## Status
Accepted

## Context
Session 026 expands the combat core with contextual turn resolution.

## Decision
- Introduce `CombatTurnContext` into `CombatSystem`.
- Keep legacy `ResolveTurn(int, int)` API for compatibility.
- Add armor mitigation and critical-hit multiplier pipeline.
- Add session artifacts (contract, schema, migration, serializer, tests).

## Consequences
- Combat core supports richer scenarios while preserving earlier session behavior.
- Future mission/combat features can inject context without rewiring previous sessions.
