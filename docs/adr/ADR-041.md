# ADR-041: Mission Reachability Validation

## Status
Accepted

## Context
Session 041 introduces mandatory graph reachability checks before mission runtime starts to prevent orphaned mission nodes.

## Decision
- Introduce Session041 reachability contract/schema/migration/serializer/profile.
- Add ReachabilityValidator and integrate it into mission validation pipeline.
- Add MissionFactory fallback path for invalid mission graphs.

## Consequences
- Mission definitions with unreachable nodes are detected before execution.
- Runtime can recover via deterministic fallback mission when source content is invalid.
