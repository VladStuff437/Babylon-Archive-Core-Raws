# ADR-034: Economy V1 Runtime and Faction Pricing

## Status
Accepted

## Context
Session 034 establishes an economy baseline with inflation, faction-specific price modifiers, and transaction synchronization with world state.

## Decision
- Introduce Session034 economy contract/schema/migration/serializer.
- Extend EconomyState with inflation index, faction modifiers, and buy/sell quote APIs.
- Add WorldEconomySynchronizer faction transaction flow.
- Add session-034 balance profile and control profile for economy tuning.

## Consequences
- Economy is now configurable through balance data and reputation-aware pricing.
- Mission/economy interactions are testable via runtime APIs.
