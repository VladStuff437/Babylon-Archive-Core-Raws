# ADR-043: Cycle Safety and Deterministic Seeded Generation

## Status
Accepted

## Context
Session 043 extends mission safety with cycle validation and introduces canonical archive addressing plus deterministic seed-driven level generation.

## Decision
- Introduce Session043 cycle-safety contract/schema/migration/serializer/profile.
- Add CycleSafetyValidator and enforce unsafe-cycle rejection before runtime start.
- Add ArchiveAddress/ArchiveSeed canonicalization and deterministic LevelGenerator with room archetypes.

## Consequences
- Mission graphs with unsafe closed cycles are rejected or redirected to fallback runtime path.
- Generated room layouts become reproducible for identical archive addresses and world seeds.
