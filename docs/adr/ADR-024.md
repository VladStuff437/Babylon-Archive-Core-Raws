# ADR-024: Character Attributes Foundation

## Status
Accepted

## Context
Session 024 formalizes character attributes as part of the real-time gameplay contour.

## Decision
- Extend `PlayerState` with Strength, Agility, Intellect, Vitality.
- Add clamped range validation for all character attributes.
- Add dedicated session artifacts (contract, schema, migration, serializer, tests).

## Consequences
- Character progression mechanics can be layered without changing core combat loop.
- Attribute values stay within deterministic and valid bounds.
