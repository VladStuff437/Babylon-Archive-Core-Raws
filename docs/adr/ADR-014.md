# ADR-014: Data Migration Pipeline Consolidation

**Статус**: Accepted
**Дата**: Сессия 014
**Контекст**: ADR-011..013

## Решение
Консолидировать migration pipeline в отдельный контракт и явный migration шаг версии 14.

## Альтернативы
1. Продолжать миграции без отдельного контракта: отклонено.
2. Перенести миграции в UI слой: отклонено.

## Риски
- Рост сложности цепочки миграций.
- Возможная деградация производительности при длинных апгрейдах.
