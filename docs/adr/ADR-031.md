# ADR-031: Damage Formulas and Mission Runtime Foundation

## Status
Accepted

## Context
Сессия 031 стабилизирует боевую математику и вводит базовый контракт миссионного ядра: формулы урона, таблицы редкости лута, экономический контур и evaluator переходов.

## Decision
- Добавить `DamageCalculator` с защитой граничных случаев.
- Добавить `DropResolver` и загрузку баланс-таблиц через `BalanceTableLoader`.
- Ввести `EconomyState`, `WorldState`, `MissionDefinition`, `MissionNode`, `TransitionEvaluator`.
- Добавить session-артефакты S031: contract/schema/migration/serializer/profile/tests.

## Consequences
- Урон и лут рассчитываются предсказуемо и параметризуемо.
- Миссионный runtime получил формальный базовый каркас.
