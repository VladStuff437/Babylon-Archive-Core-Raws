# ADR-046: ArchiveAddress Model and Hierarchical Seed Navigation

## Status
Accepted

## Context
Session 046 finalizes ArchiveAddress as a navigable model and strengthens hierarchical seed derivation for archive generation.

## Decision
- Introduce Session046 archive-address contract/schema/migration/serializer/profile.
- Extend ArchiveAddress with path/root/navigation helpers.
- Add ArchiveSeed hierarchical derivation API and seed golden tests for address model behavior.

## Consequences
- Archive addressing becomes easier to validate and navigate in runtime and tests.
- Parent/child address transitions and seeds stay deterministic under replay.
