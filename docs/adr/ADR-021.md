# ADR-021: Camera System Foundation (S021)

## Status
Accepted

## Context
Block 03 milestone requires a robust camera system supporting multiple modes (Follow, Aim, Inspect) for gameplay and device-aware profile selection. This ADR introduces the contract, schema, runtime, migration, serializer, control profile, and tests for the camera system.

## Decision
- Add `Session021CameraContract` for camera state and modes.
- Add `session-021.schema.json` for camera state validation.
- Add `CameraStateManager` for runtime camera state management.
- Add `Migration_021` for state migration.
- Add `Session021Serializer` for serialization.
- Add `session-021.profile.json` and update manifest for camera controls.
- Add runtime and smoke tests for camera system.

## Consequences
- Camera system is now extensible for gameplay and device-aware input.
- Enables further development of interactions, inventory, and combat in S022–S023.
