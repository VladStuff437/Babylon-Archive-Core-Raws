# ADR-030: Enemy State Machine Expansion

## Status
Accepted

## Context
Сессия 030 расширяет поведенческий контур врагов: нужны промежуточные состояния и детерминированные переходы при потере цели, низком здоровье и ограничении leash-distance.

## Decision
- Расширить `AIState` состояниями `Alert`, `Investigate`, `ReturnToPost`.
- Добавить в `AIStateMachine` метод `UpdateEnemyState(EnemyStateInput)`.
- Зафиксировать session-артефакты S030: contract/schema/migration/serializer/profile/tests.

## Consequences
- Поведение врага стало более выразительным и тестируемым.
- Создан фундамент для балансировки боевых паттернов в S031/S032.
