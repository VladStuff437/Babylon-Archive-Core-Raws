# ADR-015: Runtime Event Taxonomy Baseline

**Статус**: Accepted
**Дата**: Сессия 015
**Контекст**: ADR-014

## Решение
Ввести базовую таксономию событий runtime (domain/category/severity) и синхронизировать SessionEvents с новыми этапами.

## Альтернативы
1. Оставить flat-список событий: отклонено.
2. Использовать внешнюю event taxonomy библиотеку: отклонено для текущей стадии.

## Риски
- Переименование событий может ломать совместимость логов.
- Требуется дисциплина версионирования taxonomy.
