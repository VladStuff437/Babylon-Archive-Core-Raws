# ADR-028: Tick-Based Auto-Attack

## Status
Accepted

## Context
Сессия 028 переводит автоатаку на тиковую модель с контролем интервала и безопасным стопом, сохраняя совместимость с базовым API.

## Decision
- Расширить `AutoAttackController` интервалом атак (`AttackIntervalTicks`) и тик-методом `Tick()`.
- Ввести миссионную блокировку для подавления ударов в переходах.
- Сохранить `Start(string)` и `SafeStop()` как совместимый слой.
- Добавить session-артефакты: contract/schema/migration/serializer/control profile/tests.

## Consequences
- Автоатака стала детерминированной и тестируемой.
- Безопасная остановка гарантирует консистентное состояние таргета.
