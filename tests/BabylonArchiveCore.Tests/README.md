# BabylonArchiveCore.Tests

Тесты проекта Babylon Archive Core.

## Категории тестов
- **Unit** — изолированные тесты доменных моделей и логики
- **Integration** — тесты взаимодействия слоёв
- **Smoke** — быстрые проверки базовой работоспособности
- **Golden** — сравнение с эталонными данными (seed replay)
- **Property-based** — генеративные тесты инвариантов
- **Regression** — воспроизведение найденных багов
