# V1 PLAYABLE CRITERIA — Babylon Archive Core

> Frozen: Session 1 | 2026-03-21

---

## MUST HAVE — Без этого v1 не считается playable

| # | Критерий | Проверка |
|---|----------|----------|
| 1 | Игра запускается без краша | Запуск → главная сцена < 5 сек |
| 2 | Алан просыпается в капсуле | Стартовая камера + выход по кнопке |
| 3 | Можно ходить по A-0 в 3D | WASD + mouse look, коллизии работают |
| 4 | Биометрическая станция работает | Подход + E → подтверждение имени |
| 5 | Логистический терминал работает | Подход + E → стартовые предметы |
| 6 | Дрон отвечает | Минимум 3 реплики |
| 7 | C.O.R.E. активируется | Сцена + переход к OperationAccess |
| 8 | Терминал операций показывает карточку | Видна первая операция |
| 9 | Обзорная галерея показывает шахту | Виден Хард-Архив внизу |
| 10 | Вход в Хард-Архив заблокирован | Сообщение о блокировке |
| 11 | Протокол Ноль разблокирован | Финальное сообщение пролога |
| 12 | Стабильность | Нет крашей при полном прохождении |
| 13 | Повторный запуск | Перезапуск не ломает состояние |
