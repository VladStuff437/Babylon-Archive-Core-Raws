## INCLUDE — Что входит в v1

### Сцена
- SCN_A0_INITIATION — единственная играбельная сцена
- Палата Инициализации A-0 (INIT_CHAMBER_A0)
- 11 зон хаба (см. A0_Production_Bible.md)

### Игрок
- Управление Аланом в 3D
- Переключение камеры 3D ↔ 2.5D
- Движение WASD + мышь
- Взаимодействие по кнопке

### Интерактивы
- Капсула пробуждения (выход)
- Биометрический терминал (идентификация)
- Логистический терминал (снаряжение)
- Ниша дрона (первый NPC-контакт)
- C.O.R.E. (центральный сюжетный узел)
- Терминал операций (древо миссий)
- Обзорная галерея (вид на Хард-Архив)
- Вход в Хард-Архив (заблокирован, виден)

### Прогрессия
- Имя, уровень, XP
- 5 базовых статов (Analysis, Influence, Tech, Arcane, Endurance)
- Стартовый инвентарь (3 предмета)
- Намёк на компетенции (не полная система)

### Контент
- JSON-файлы для сцены, зон, объектов, диалогов, триггеров, целей
- Пролог проходится за 8–15 минут
- Тексты на русском (игровой язык v1)

### Сохранения
- Сохранение/загрузка текущего прогресса пролога
- Защита от битых сейвов

### Отладка
- Debug overlay: зона, триггеры, фокус
- Телепорт по зонам

## Критерий «готово»

v1 считается завершённым, когда игрок может:
1. Запустить игру
2. Проснуться в капсуле
3. Пройти по A-0 в правильном порядке
4. Увидеть вход в Хард-Архив
5. Разблокировать первую операцию
6. Игра стабильно работает без краша
